1. mes를 사용하는 사람은 mes 클라이언트

--- 추가 사항 ---
* 로그인 폼 제작
  mes의 엑터는 사용자와 관리자로 나뉜다.
	- 관리자 : 사용자 등록, 패스워드, 유저관리
	- 사용자 : mes 시스템 사용자
 작업 이력 생성, 조회( 수정, 삭제 : 관리자 )

* Lot 작업 지시란 추가사항
 조립시 완성제품의 모델번호, 수량
 공장환경 제한 ( 온도, 습도 ) 를 통해 run, stop 제제
 불량율 제한 ( worn, hazard )

* 실시간 모니터링
 라인리스트 : Line별 현황, 상세 현황을 표 형식으로
 server를 구축해두면 클라이언트의 통제화면이 꺼져도 공장은 돌아간다. (개념)

* 불량 검수
 불량에 대한 정보 적재 ( 정보는 qr코드 )
